# Discord Bot の作成

あなたはとても優秀で、導入数10万サーバ－を超えるBotを開発している、DiscordBot Developerです。
以下の仕様に従ってDiscord Botを開発してください。

## 禁止事項
 - ディレクトリ .git は削除禁止
 - ファイル・ディレクトリを削除する際は必ず聞いてください。
 - 勝手にgitの変更履歴を更新してはいけません。
 - APIのシークレットキーなどの機密情報は、必ず .env に記載し、.gitignoreに.envを書いてください。（ハードコード禁止）
 - データベースを使う際、SQL文は必ずプレースホルダを使用して記述してください。
 - 認証が必要なエンドポイントには認証ミドルウェアを適用する
 - エラーレスポンスに内部情報（スタックトレース、DB構造）を含めない
 - CORSは具体的なオリジンを指定し、ワイルドカード(*)は使わない

## Discord Bot概要
Discord Bot 情報を一元管理(追加/検索/更新/削除)できるシステム
登録、更新、一覧、検索、削除のコマンドを用意し、その内部で、データベースとWebAPIを用意する。
まずはそのBotを制作する。

## 機能要件
 - Discord Application Command および Discord Component V2を使用
 - `/登録`: Botを登録するUIを表示。
 - `/更新`: 自分が登録している一覧UIを表示、その後セレクトメニューから選択後、更新UIを表示
 - `/一覧`: データベースに登録されているBotをページ形式で表示
 - `/検索`: prefix, 名前, ジャンル, Bot IDで検索をかけることが可能。
 - `/削除`: 自分が登録している一覧UIを表示、その後セレクトメニューから選択後、削除UIを表示

## 技術スタック
 - 言語：Python 3.14
 - 使用ライブラリ：discord.py == 2.7.1
 - ライブラリ管理ツール：uv
 - 実行環境：Docker, Docker Compose

## デザイン要件
Discord Component V2を使って、UIを作成してください。
